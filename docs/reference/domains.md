# Domains API

Domain APIs are intended to be used in this order:

1. compute/add/rasterize via `atlas.wind` or `atlas.landscape`
2. materialize into active atlas stores
3. continue from `atlas.wind.data` / `atlas.landscape.data`

Use this page for exact signatures and raised-error details.
